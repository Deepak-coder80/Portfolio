üí° Usage Examples

Interactive Mode (Recommended)

# Launch interactive wizard ./iam-role-cloner clone # Dry-run with verbose output ./iam-role-cloner clone --dry-run --verbose

Command Line Automation

# Clone with all parameters specified ./iam-role-cloner clone \ --source-profile dev \ --dest-profile prod \ --source-pattern "dev_" \ --dest-pattern "prod_" \ --dry-run # List roles in a profile ./iam-role-cloner list --profile dev --pattern "app" # Show version information ./iam-role-cloner version --detailed

Environment Migration Examples

# Development to Production ./iam-role-cloner clone \ --source-profile development \ --dest-profile production \ --source-pattern "dev-" \ --dest-pattern "prod-" # Staging to Live ./iam-role-cloner clone \ --source-profile staging \ --dest-profile live \ --source-pattern "staging-" \ --dest-pattern "live-"

Role Discovery and Inspection

# List all roles ./iam-role-cloner list --profile dev # List roles with pattern filter ./iam-role-cloner list --profile prod --pattern "app" --sort # Detailed role information ./iam-role-cloner list --profile staging --details
